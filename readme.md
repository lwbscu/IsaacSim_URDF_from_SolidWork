A lightweight, bilingual (English/Chinese) PyQt5 GUI tool designed to sanitize and restructure URDF files and their corresponding assets (especially those exported from SolidWorks). It ensures a seamless, error-free import process into **NVIDIA Isaac Sim**, completely eliminating the notorious `Null Prim` crashes caused by strict USD naming conventions and ROS absolute path dependencies.

## Key Features

- **Naming Sanitization**: Automatically detects and replaces illegal characters (Chinese, spaces, hyphens `-`) in physical files, folders, and inner URDF attributes (Links/Joints/Meshes) to meet the strict USD standard.
- **Path Restructuring**: Strips `package://` absolute paths commonly generated by ROS/ROS2 exporters and safely converts them into robust relative paths (`../meshes/xxx.STL`).
- **Joint Topology Analyzer**: Deeply parses the URDF XML tree to count and categorize joint types (Revolute, Prismatic, Fixed). Helps developers catch misconfigured "Fixed" wheels or grippers before importing.
- **Bilingual UI**: Seamlessly switch between English and Simplified Chinese on the fly.
- **Zero-Configuration Startup**: The provided `run.sh` script automatically checks for dependencies, interactively installs missing ones (`PyQt5`), and mutes harmless underlying Linux GUI warnings for a clean terminal experience.

## Quick Start

### Prerequisites
- Ubuntu / Linux environment (or WSL2)
- Python 3.x

### Installation & Run
Simply clone the repository and run the startup script:

```bash
git clone [https://github.com/YourUsername/IsaacSim-URDF-Cleaner.git](https://github.com/YourUsername/IsaacSim-URDF-Cleaner.git)
cd IsaacSim-URDF-Cleaner
chmod +x run.sh
./run.sh
```

(Note: If PyQt5 is not installed, the script will prompt and safely install it for you using pip.)

## How to Use

Launch the tool using ./run.sh.
Select Target Folder: Browse and select the root directory of your robot assets.
Select Original URDF: Browse and select the specific .urdf file (the root folder path will auto-fill if standard structure is detected).
Input New Name: Provide a clean, USD-compliant name (e.g., robot_v2 - strictly English/underscores, no .urdf suffix).
Click [2. One-Click Clean & Rebuild].
Open NVIDIA Isaac Sim -> File -> Import -> select your newly generated .urdf file. Enjoy the flawless import!
